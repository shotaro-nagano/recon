# タイプ画像 キャラデザプロンプト集 (ChatGPT画像生成用)

prompts/imagegen.md の規定に基づき生成。**ChatGPTにコードブロックをそのまま貼るだけで使える**。
各タイプ1プロンプト。α/Ωは同一プロンプト内の差分指定、SKIN A/B/C は3回生成して使い分ける。

- 生成した画像は `assets/types/{コードネーム}_{a|b|c}_{alpha|omega}.png` で保存
- まず自分のタイプの3スキンだけ作れば運用開始できる(8タイプ×3スキン×2=全48枚)
- 画像はアプリ内表示専用(プライバシー規定)。実在選手・既存作品の名前は使わない
